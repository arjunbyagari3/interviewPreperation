They are enclosed in angle brackets, such as `<p>` for a paragraph or `<a>` for a link. Tags typically come in pairs, with an opening tag (e.g., `<p>`) and a closing tag (e.g., `</p>`), although some tags are self-closing (e.g., `<img />`).    Attributes provide additional information about an element and are included within the opening tag. They consist of a name and a value, separated by an equals sign. For example, in the tag `<a href="https://www.example.com">`, `href` is the attribute name and `"https://www.example.com"` is the attribute value. Attributes can be used to specify properties such as links, styles, classes, and more.4) How do you create a hyperlink in HTML?To create a hyperlink in HTML, you use the `<a>` (anchor) tag. Here is an example of how to create a hyperlink:```html<a href="https://www.example.com">Visit Example</a>```In this example, the text "Visit Example" will be displayed as a clickable link that directs the user to "https://www.example.com" when clicked.
